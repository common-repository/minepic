Replace wordpress avatars with minecraft skins.

== Description ==

With the Minepic plugin you can easily replace the wordpress avatars with the head of the minecraft skins of your users, using https://minepic.org

Features:

* Minecraft avatars for commenters and users
* BuddyPress support
* Using HTTPS

Note: This plugin is a modification of the Minotar plugin. The original code was written by clone1018 and edvindAll!

== Installation ==

1. Upload the 'minepic' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How do I change the avatars back? =

Either you change the default avatar in Settings - Discussion on the admin panel or you just delete the plugin.
